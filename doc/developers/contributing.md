# Contributing

[Home](../README.md) → [Developers](contributing.md) → Contributing

Stepflow is pre-1.0 and under active development. Contributions are welcome.

## Local setup

```bash
npm install
```

## Quality checks

```bash
npm run lint
npm run type-check
npm run test
```

Notes:

- The repository uses ESLint, Prettier, and Vitest.
- Please open an issue or discussion for significant changes.
