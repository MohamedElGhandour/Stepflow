# Tooltip: Options

[Home](../../README.md) → [Features](../steps/overview.md) → [Tooltip](overview.md) → Tooltip: Options

The tooltip has no dedicated configuration object. Its content is driven by step content, and its layout depends on `progress.position`.

## Relevant options

- `steps[].content` (see `Steps` feature)
- `progress.position` (see `Progress Indicator` feature)

Notes:

- Tooltip dimensions and spacing are controlled by CSS in `src/styles/stepflow.scss`.
