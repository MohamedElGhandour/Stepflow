Stepflow

Welcome to Stepflow

Your lightweight, simple, and customizable guided tour solution.

Introduction

Stepflow makes it easy to create guided tours for your web applications. With an emphasis on simplicity and customization, you can onboard your users quickly and efficiently.

Features

Lightweight

Minimal code and fast performance for a seamless user experience.

Simple

Integrate Stepflow with a straightforward API and minimal effort.

Customizable

Tailor every element to match your brand and design preferences.

Installation

You can install Stepflow via NPM or include it directly via CDN.

Via NPM

npm install stepflow

Via CDN

<link rel="stylesheet" href="https://unpkg.com/stepflow/dist/stepflow.css">
<script src="https://unpkg.com/stepflow/dist/stepflow.js"></script>
